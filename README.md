## Coding Convention
+ **Naming Rule**
  + **Class** -> Pascal Case　　　ex) HelloWorld
  + **Method** -> camelCase　　　ex) helloWorld
  + **Variable** -> camelCase　　　ex) helloWorld
   + (Android) button -> snake_case　　　ex) login_btn
